Website-to-Telegram Integration

Project Overview

Type: Web Integration

Development Approach: Vibe Coding with Claude & Cursor

Role: Solution Designer & AI Director

Problem & Solution

Problem: Missing contact form submissions or delayed responses due to infrequent email checking.

Solution: Real-time notifications via Telegram when someone submits the contact form.

Impact: Improved response time to potential clients and collaborators, no need to constantly check email.

Technical Implementation

  • Backend Service

    Lightweight API service to receive form submissions and forward them to Telegram

  • Telegram Bot

    Custom bot that sends formatted messages directly to my phone

  • Security

    Protection against cross-origin requests and secure configuration

  • Deployment

    Containerized service for easy deployment and scaling

AI-Assisted Development

This entire solution was developed using Vibe Coding principles. I defined the requirements and architecture, then used Claude to generate the initial backend and JavaScript integration. After reviewing the code, I used Cursor to refine implementation details and add security features. This approach reduced development time from days to hours.
